# cns-wc26-resource

本仓库用于归档和发布多个独立的 HTML 页面。这些页面不针对任何特定网站或域名，仅作为通用资源存放。

## 项目简介

cns-wc26-resource 是一个静态 HTML 页面集合仓库。页面内容彼此独立，可根据需要单独访问或引用。仓库旨在提供一个干净、无干扰的页面归档环境。

## 目录说明

```
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的公共资源（CSS、JS、图片等）
└── README.md       # 本文件
```

- **pages/**：每个 HTML 文件是一个独立页面，命名清晰，便于识别。
- **assets/**：包含页面所需的样式、脚本及媒体文件，按需组织。

## 页面归档说明

- 所有页面均为标准 HTML 格式，可直接在浏览器中打开。
- 页面之间无依赖关系，可独立运行或嵌入其他环境。
- 归档内容不包含任何外部链接或商业信息，保持纯粹。

## 使用方式

1. 克隆或下载本仓库。
2. 打开 `pages/` 目录下的任意 HTML 文件即可查看。
3. 如需部署，可将整个仓库上传至静态托管服务。
